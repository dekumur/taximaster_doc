Минимальные системные требования:

Компонент Характеристика
Процессор Core i3 (4 ядра)
Оперативная память DDR4(5) 6-8 Гбайт
SSD диск от 120Gb для установки ОС и программного комплекса Такси-Мастер
Жесткий диск(для хранения резервных копий) 500Gb (с учетом регулярного резервного копирования и при условии хранения копий базы данных за последние 90 дней)
Источник бесперебойного питания Обязательно наличие источника бесперебойного питания
Операционная система Только оригинальная копия 64-х битной операционной системы MS Windows 10 Pro, Windows 11 Pro, либо линейка Windows Server, начиная с Windows Server 2016.
Монитор

При соотношении сторон монитора 4:3 разрешение должно быть не менее 1152x864.
При соотношении сторон монитора 16:10 разрешение должно быть не менее 1440x900.
При соотношении сторон монитора 16:9 разрешение должно быть не менее 1600x900.
